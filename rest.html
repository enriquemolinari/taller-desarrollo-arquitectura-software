¿Qué es un Servicio HTTP?

Denominamos Servicio HTTP o Servicio WEB (o mal llamado a veces Servicio REST), simplemente a un servicio expuesto en un Servidor Web que como respuesta retorna JSON o XML (en lugar de una pagina web).
